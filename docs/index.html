LiveQuery

Real-time SQL subscriptions for PostgreSQL. Subscribe to queries, get live updates via NATS.

GitHub Download

Quick Example

// Subscribe to pending orders const res = await nats.request('livequery.orders.subscribe', { subscription_id: 'orders', query: 'SELECT * FROM orders WHERE status = \'pending\'', mode: 'events' }); // Receive live updates nats.subscribe(res.subject, (msg) => { console.log('Update:', msg.events); });

Features

Plain SQL

No new languages. Write standard SQL queries and subscribe to their results.

Low Latency

Native PostgreSQL WAL streaming. Sub-10ms change detection.

Shared Queries

Identical queries share one snapshot. Efficient memory usage at scale.

Two Modes

Events mode for differential updates. Snapshot mode for full data.
